# Roadmap

## Versión 0.5.1

- Corrección de errores.
- Mejoras en la documentación.
- Optimización del instalador.
- Optimización del desinstalador.

---

## Versión 0.6.0

- Sistema de actualización.
- Detección automática de dependencias.
- Configuración inicial guiada.
- Mejor integración con MPV.
- Mejor soporte para YouTube.

---

## Versión 0.7.0

- Sistema de plugins.
- Radio por Internet.
- Podcasts.
- Mejor administración de listas IPTV.

---

## Versión 0.8.0

- Compatibilidad con Debian.
- Compatibilidad con Arch Linux.
- Mejor soporte para Alpine Linux.
- Mejoras de rendimiento.

---

## Versión 1.0.0

Primera versión estable de Open Media Center.

Objetivos:

- Instalación sencilla.
- Sistema modular estable.
- Documentación completa.
- Código optimizado.
- Proyecto listo para la comunidad.
